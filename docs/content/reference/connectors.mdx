# Channel Connectors

Synapse supports 40 channel connectors for ingesting data from external platforms. Each connector polls a source for new events and routes them through your ingest and update pipelines.

## Using Connectors

Connectors are configured inside `channel` blocks in your `.mnm` file:

```mnm
channel my_feed {
  source: connector_name
  config {
    token: env("API_TOKEN")
  }
  poll_interval: 5m

  on message -> ingest(content: string) {
    content |> extract() |> store()
  }
}
```

## Event Types

| Event | Description |
|-------|-------------|
| `message` | New message received |
| `edit` | Existing message was edited |
| `delete` | Message was deleted |
| `reaction` | Reaction added to a message |
| `thread_reply` | Reply in a thread |
| `file_upload` | File was uploaded |
| `member_join` | User joined |
| `member_leave` | User left |
